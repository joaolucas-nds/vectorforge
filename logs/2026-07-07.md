## Aprendizados / armadilhas
- **Nova Armadilha 8 (CONTEXT.md)**: bundle IIFE não vaza funções para `window` automaticamente — qualquer handler novo referenciado via `onclick=""` inline (estático ou gerado em template string) precisa ser adicionado explicitamente ao `Object.assign(window, {...})` em `main.js`, ou o clique silenciosamente não faz nada, sem erro no console. Esse é o tipo de bug que só um teste de INTERAÇÃO (clicar de verdade) pega — uma comparação estática de texto não pegaria.
- **Extração automatizada > retype manual para refactors grandes**: usar um parser (mesmo que simples, de contagem de chaves) para mover ~1500 linhas de código entre arquivos elimina uma classe inteira de erro de transcrição que seria fácil de cometer e difícil de notar numa revisão visual.
- **Comparação byte-a-byte via browser headless é o teste mais forte disponível para "este refactor não mudou nada"**: mais forte que ler o diff do código (que não pega "esqueci de importar X", já que isso só aparece em runtime) e mais forte que abrir manualmente e clicar em alguns botões (que não cobre as 49 combinações). Vale generalizar para o resto de F3.
